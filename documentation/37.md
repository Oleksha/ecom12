# Часть 37. Атрибуты продукта (I) | Создать таблицу и вставить атрибуты с помощью Seeder
С этого момента мы начнем работу над модулем атрибутов товара — важнейшей функцией любого интернет-магазина.
### Что такое атрибуты продукта?
Атрибуты товара — это конкретные характеристики или варианты товара, такие как размер, цвет, артикул, цена и наличие на складе. Например, футболка может быть маленького, среднего и большого размера — каждый со своей ценой, наличием на складе и артикулом.

Эти атрибуты помогают пользователям выбирать именно тот вариант продукта, который они хотят купить, а администратору — эффективно управлять запасами.

В этой части мы настроим структуру базы данных для атрибутов и вставим образцы записей с помощью Seeder. (Позже мы вставим атрибуты из пользовательского интерфейса панели администратора)

**Что мы будем делать в этом видео**:
- Создадим таблицу products_attributes с помощью миграции.
- Добавим несколько атрибутов продукта через Seeder.
- Создадим модель для атрибутов продукта
- Подготовимся к следующей части: добавление атрибутов через панель администратора
## Шаг 1. Создайте таблицу products_attributes с помощью миграции
Выполните команду artisan: 
```
php artisan make:model ProductsAttribute -m
```
Теперь обновите файл миграции следующим образом:
```php
Schema::create('products_attributes', function (Blueprint $table) {
    $table->id();
    $table->foreignId('product_id')->constrained()->onDelete('cascade');
    $table->string('size');
    $table->string('sku')->unique();
    $table->decimal('price', 10, 2);
    $table->unsignedInteger('stock');
    $table->unsignedInteger('sort')->default(0);
    $table->tinyInteger('status')->default(1)->comment('1: Active, 0: Inactive');
    $table->timestamps();
});
```
Теперь перенесем таблицу:
```
php artisan migrate
```
## Шаг 2. Написание Seeder: ProductsAttributesTableSeeder
Сгенерируйте файл Seeder:
```
php artisan make:seeder ProductsAttributesTableSeeder
```
Это создаст:
```
database/seeders/ProductsAttributes TableSeeder.php
```
Обновите файл, как показано ниже:
```php
namespace Database\Seeders;

use App\Models\ProductsAttribute;
use Illuminate\Database\Seeder;

class ProductsAttributesTableSeeder extends Seeder
{
    /**
     * Run the database seeds.
     */
    public function run(): void
    {
        $productAttributesRecords = [
            [
                'product_id' => 1,
                'sku' => 'RCT001-S',
                'size' => 'Small',
                'price' => 1500,
                'stock' => 10,
                'sort' => 1,
                'status' => 1,
            ],
            [
                'product_id' => 1,
                'sku' => 'RCT001-M',
                'size' => 'Medium',
                'price' => 1600,
                'stock' => 20,
                'sort' => 2,
                'status' => 1,
            ],
            [
                'product_id' => 1,
                'sku' => 'RCT001-L',
                'size' => 'Large',
                'price' => 1700,
                'stock' => 10,
                'sort' => 3,
                'status' => 1,
            ],
        ];
        ProductsAttribute::insert($productAttributesRecords);
    }
}
```
## Шаг 3. Обновление DatabaseSeeder.php
Откройте database/seeders/DatabaseSeeder.php и включите новый seeder:
```php
public function run(): void
{
    $this->call(AdminsTableSeeder::class);
    $this->call(CategoryTableSeeder::class);
    $this->call(ColorTableSeeder::class);
    $this->call(ProductsAttributesTableSeeder::class);
}
```
## Шаг 4. Запустите Seeder
Теперь выполните эту команду, чтобы вставить образец данных:
```
php artisan db:seed --class ProductsAttributesTableSeeder
```

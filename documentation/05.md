# Часть 5. Создание учетных данных администратора с помощью Seeder
В этой части мы сгенерируем запись администратора с именем пользователя и паролем, используя **seeder**.
## Шаг 1. Создайте файл **AdminsTableSeeder**:
Сначала мы сгенерируем **seeder** для вставки учетных данных администратора.
Выполните следующую команду: 
```
php artisan make:seeder AdminsTableSeeder
```
Она создаст файл **AdminsTableSeeder.php** по адресу:
```
database/seeders/
```
Теперь откройте файл **AdminsTableSeeder.php** и добавьте следующий код.
```php
namespace Database\Seeders;

use App\Models\Admin;
use Illuminate\Database\Console\Seeds\WithoutModelEvents;
use Illuminate\Database\Seeder;
use Hash;

class AdminsTableSeeder extends Seeder
{
    public function run(): void
    {
        $password = Hash::make('123456');
        $admin = new Admin;
        $admin->name = 'Oleg Deryabin';
        $admin->role = 'admin';
        $admin->mobile = '79061291066';
        $admin->email = 'admin@mail.ru';
        $admin->password = $password;
        $admin->status = 1;
        $admin->save();
    }
}
```
Мы генерируем хешированный пароль для администратора с помощью функции **Hash::make**.

Она преобразует «123456» в безопасный хеш-формат для хранения в базе данных.
## Шаг 2. Обновите файл **DatabaseSeeder.php**:

Обновите файл **DatabaseSeeder.php**, расположенный в **database/seeders/**, чтобы вызвать класс **AdminsTableSeeder**.
```php
public function run(): void
{
    $this->call(AdminsTableSeeder::class);
}
```
## Шаг 3. Запустите **Seeder** для вставки учетных данных администратора:
Выполните следующую команду, чтобы вставить данные администратора в базу данных:
```
php artisan db:seed --class=Admins TableSeeder
```
После выполнения этой команды эти данные будут вставлены в таблицу admins, что позволит вам используя их войти в панель управления.

# Часть 51. Интеграция HTML-шаблона в настройку домашней страницы электронной коммерции на Laravel с несколькими поставщиками.
В этой части мы преобразуем HTML-шаблон нашего мультивендорного сайта электронной коммерции (загруженный в предыдущей части) в структуру на базе Laravel и создадим домашнюю страницу.
## Шаг 1. Копирование папок статических ресурсов:
Мы скопируем следующие папки из HTML-шаблона в нашу структуру Laravel:
- img → /public/front/images/
- mail → /public/front/mail/
- css → /public/front/css/
- js → /public/front/js/
- lib → /public/front/lib/
## Шаг 2. Создайте файлы макетов Blade:
Создайте папку layout в **/resources/views/front/**.

Внутри нее создайте следующие файлы:
- layout.blade.php
- header.blade.php
- footer.blade.php
- styles.blade.php
- scripts.blade.php

Мы скопируем основную HTML-структуру из файла **index.html** шаблона в **layout.blade.php**. Затем извлечём повторно используемые элементы, такие как заголовок, нижний колонтитул, стили и скрипты, в соответствующие файлы.

**Пример структуры blade**:
```html
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="utf-8">
    <title>Laravel 12 Multi-Vendor E-commerce Template by SiteMakers.in</title>
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="keywords" content="Laravel 12, Multi-Vendor E-commerce, Laravel E-commerce Template, Bootstrap Laravel, Laravel Admin Panel, Laravel Marketplace, SiteMakers, Stack Developers">
    <meta name="description" content="Download the SiteMakers Laravel 12 Multi-Vendor E-commerce Template – a clean and scalable Laravel project built with Bootstrap, Blade, and a powerful admin panel. Ideal for learning and prototyping e-commerce websites.">
    <!-- Favicon -->
    <link href="{{ asset('front/img/favicon.ico') }}" rel="icon">
    @include('front.layout.styles')
</head>
<body>
    @include('front.layout.header')
    @yield('content')
    @include('front.layout.footer')
    @include('front.layout.scripts')
</body>
</html>
```
## Шаг 3. Создайте файл index.blade.php:
Создайте **index.blade.php** в папке: **/resources/views/front/**

Скопируйте среднее содержимое из **index.html** HTML-шаблона в этот файл.
```html
@extends('front.layout.layout')
@section('content')
    <!-- Carousel Start -->
    Код файла
    <!-- Vendor End -->
@endsection
```
## Шаг 4. Создать IndexController:
Создайте новый контроллер внутри папки Front: 
```
php artisan make:controller Front/IndexController
```
Это помогает разделить логику Front и Admin.
## Шаг 5. Добавить маршрут для домашней страницы:
Добавьте это в начало **web.php**:
```php
use App\Http\Controllers\Front\IndexController;
```
И затем определите маршрут:
```php
Route::namespace('App\Http\Controllers\Front')->group(function () {
    Route::get('/', [IndexController::class, 'index']);
});
```
## Шаг 6. Создать функцию index():
Внутри **IndexController** определите метод:
```php
public function index()
{
    return view('front.index');
}
```
Теперь ваша домашняя страница Laravel связана и будет отображаться, но изначально без стилей и изображений, что мы исправим далее.

Исправление путей к ресурсам с помощью функции Laravel **asset()**

Обновите **styles.blade.php**:
```html
<link href="{{ asset('front/lib/owlcarousel/assets/owl.carousel.min.css') }}" rel="stylesheet">
<link href="{{ asset('front/css/style.css') }}" rel="stylesheet">
```
Обновите **scripts.blade.php**:
```html
<script src="{{ asset('front/lib/easing/easing.min.js') }}"></script>
<script src="{{ asset('front/lib/owlcarousel/owl.carousel.min.js') }}"></script>
<script src="{{ asset('front/mail/jqBootstrapValidation.min.js') }}"></script>
<script src="{{ asset('front/mail/contact.js') }}"></script>
<script src="{{ asset('front/js/main.js') }}"></script>
```
Обновите пути к изображениям в **index.blade.php**:
```html
<img src="{{ asset('front/images/carousel-1.jpg') }}" alt="...">
```

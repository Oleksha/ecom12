# Часть 53. Управление баннерами на главной странице (I) | Создание данных миграции, посева и фиктивных баннеров
С этого момента мы начинаем работу над модулем динамических баннеров на главной странице, в котором баннеры будут управляться из панели администратора, а не будут жестко прописаны в HTML.
## Шаг 1: Создайте таблицу баннеров
Мы создадим новую модель и выполним миграцию с помощью команды artisan:
```
php artisan make:model Banner -m
```
Обновите файл миграции **create_banners_table.php**, используя следующую структуру:
```php
Schema::create('banners', function (Blueprint $table) {
    $table->id();
    $table->string('image');
    $table->string('type');
    $table->string('link');
    $table->string('title');
    $table->string('alt');
    $table->integer('sort');
    $table->tinyInteger('status');
    $table->timestamps();
});
```
Теперь запустите миграцию для создания таблицы:
```
php artisan migrate
```
Результат: Таблица баннеров теперь создана в базе данных.
## Шаг 2: Добавьте фиктивные изображения баннера
Создайте следующую папку: **/public/front/images/banners/**

Поместите в эту папку фиктивные изображения баннеров (например, carousel-1.jpg, carousel-2.jpg и т. д.).

Вы можете использовать изображения-слайдеры из HTML-шаблона.
## Шаг 3: Создайте сид для фиктивных баннеров
Выполните следующую команду для создания файла посева:
```
php artisan make:seeder BannersTableSeeder
```
Обновите **BannersTableSeeder.php** следующим образом:
```php
namespace Database\Seeders;

use App\Models\Banner;
use Carbon\Carbon;
use Illuminate\Database\Seeder;

class BannersTableSeeder extends Seeder
{
    /**
     * Run the database seeds.
     */
    public function run(): void
    {
        $bannerRecords = [
            ['id' => 1, 'type' => 'Slider', 'image' => 'carousel-1.jpg', 'link' => '', 'title' => 'Products on Sale', 'alt' => 'Products on Sale', 'sort' => 1, 'status' => 1, 'created_at' => Carbon::now(), 'updated_at' => Carbon::now()],
            ['id' => 2, 'type' => 'Slider', 'image' => 'carousel-2.jpg', 'link' => '', 'title' => 'Flat 50% Off', 'alt' => 'Flat 50% Off', 'sort' => 2, 'status' => 1, 'created_at' => Carbon::now(), 'updated_at' => Carbon::now()],
            ['id' => 3, 'type' => 'Slider', 'image' => 'carousel-3.jpg', 'link' => '', 'title' => 'Summer Sale', 'alt' => 'Summer Sale', 'sort' => 3, 'status' => 1, 'created_at' => Carbon::now(), 'updated_at' => Carbon::now()],
        ];
        Banner::insert($bannerRecords);
    }
}
```
## Шаг 4: Зарегистрируйте Seeder в DatabaseSeeder.php 
Откройте **database/seeders/DatabaseSeeder.php** и добавьте:
```php
public function run(): void
{
    $this->call(AdminsTableSeeder::class);
    $this->call(CategoryTableSeeder::class);
    $this->call(BrandTableSeeder::class);
    $this->call(ColorTableSeeder::class);
    $this->call(ProductsTableSeeder::class);
    $this->call(ProductsAttributesTableSeeder::class);
    $this->call(BannersTableSeeder::class);
}
```
## Шаг 5: Запустите Seeder для вставки данных
```
php artisan db:seed --class=BannersTableSeeder
```
Результат: фиктивные баннеры теперь вставлены в таблицу баннеров.

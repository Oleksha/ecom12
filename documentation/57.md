# Часть 57. Отображать избранные и новые товары на главной странице
В этой части мы расширяем функциональность нашей домашней страницы, динамически отображая:

Рекомендуемые товары — выбираются случайным образом из числа товаров, отмеченных как рекомендуемые.

Новое поступление товаров — последние 8 товаров, добавленных в каталог

Оба раздела извлекают данные о продуктах с помощью IndexService, обрабатывают логику отката изображений и визуализируются с помощью чистых фронтенд-блоков на базе Bootstrap.
## Шаг 1: Добавьте избранные Товары() в IndexService
Файл: **app/Services/Front/IndexService.php**
```php
public function featuredProducts(): array
{
    $featuredProducts = Product::select('id','category_id','product_name','product_applied_on','product_price','product_discount','final_price','group_code','main_image')
        ->with(['product_images'])
        ->where(['is_featured' => 'Yes', 'status' => 1])
        ->where('stock', '>', 0)
        ->inRandomOrder()
        ->limit(8)
        ->get()->toArray();
    return compact('featuredProducts');
}
```
## Шаг 2: Добавьте newArrival Products() в IndexService
```php
public function newArrivalProducts(): array
{
    $newArrivalProducts = Product::select('id','category_id','product_name','product_applied_on','product_price','product_discount','final_price','group_code','main_image')
        ->with(['product_images'])
        ->where('status', 1)
        ->where('stock', '>', 0)
        ->latest()
        ->limit(8)
        ->get()->toArray();
    return compact('newArrivalProducts');
}
```
## Шаг 3. Обновите IndexController.
Файл: **app/Http/Controllers/Front/IndexController.php**
```php
public function index()
{
    $banners = $this->indexService->getHomePageBanners();
    $featured = $this->indexService->featuredProducts();
    $newArrivals = $this->indexService->newArrivalProducts();
    return view('front.index')
        ->with($banners)
        ->with($featured)
        ->with($newArrivals);
}
```

## Шаг 4: Подтвердите соответствие изображения продукта
Файл: **app/Models/Product.php**
```php
public function product_images(): HasMany
{
    return $this->hasMany(ProductsImage::class)->orderBy('sort');
}
```
## Шаг 5: HTML-код Blade для избранных товаров
Файл: **resources/views/front/index.blade.php**
```html
<!-- Products Start -->
<div class="container-fluid pt-3">
    <div class="text-center mb-4">
        <h2 class="section-title px-5"><span class="px-2">Рекомендуемые продукты</span></h2>
    </div>
    <div class="row px-xl-5 pb-3">
        @foreach($featuredProducts as $product)
            @php
                $fallbackImage = asset('front/images/products/no-image.jpg');
                $image = !empty($product['main_image'])
                    ? asset('product-image/medium/' . $product['main_image'])
                    : (!empty($product['product_images'][0]['image'])

                        : $fallbackImage);
            @endphp
            <div class="col-lg-3 col-md-6 col-sm-12 pb-1">
                <div class="card product-item border-0 mb-4">
                    <div class="card-header product-img position-relative overflow-hidden bg-transparent border p-0">

                    </div>
                    <div class="card-body border-left border-right text-center p-0 pt-4 pb-3">
                        <h6 class="text-truncate mb-3">{{ $product['product_name'] }}</h6>
                        <div class="d-flex justify-content-center">

                            @endif
```
